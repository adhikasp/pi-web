# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Machine-local dev environment notes

If a `CLAUDE.local.md` file exists alongside this one, read it before starting
or restarting a dev server, or looking for logs. It's gitignored and holds
machine-specific instructions (e.g. how this particular machine already runs
pi-web, where its logs live) that don't belong in version control. If it's
missing, no such local setup applies — fall back to `npm run dev` and standard
tooling.
